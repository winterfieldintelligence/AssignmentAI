# Assignment AI - Google Play Store Listing (Draft)

## App Name
Assignment AI

## Short Description (<=80 chars)
Create handwritten-style assignments from your own handwriting.

## Full Description
Assignment AI helps you turn your handwriting sample into clean, readable, handwritten-style
assignment pages. Upload or select your handwriting and a background page, type the content you
want, and generate realistic output in seconds.

Key features:
- Handwriting-style rendering that looks like real notebook writing.
- Use your own handwriting sample for personalized output.
- Optional font matching for quick results.
- Background page detection for margins and line spacing.
- Export and share generated pages as images.
- Works offline for most features; optional backend support for font generation.

Use cases:
- Personal notes and study material.
- Practice and formatting for assignments.
- Converting typed notes into handwriting-style pages.

Important: Please use this app responsibly and comply with your school or institution’s
academic policies.

## Data Safety (Play Console - Summary)
Collected data:
- User-provided images (handwriting samples and background pages).
- User-entered text to render assignments.

Purpose of collection:
- Generate handwriting-style output and match fonts.
- Provide core app functionality.

Data sharing:
- Data is not sold.
- Data is only shared with backend services required for font generation or style extraction.

Data retention:
- Content stays on the user’s device unless uploaded to a backend for processing.
- Backend files may be stored temporarily to complete processing.

Security:
- Data is transferred over network when backend features are used.
- Users can delete content by deleting files or uninstalling the app.

## Permissions Explanation
- INTERNET: to download the font pack and use backend services.
- READ_MEDIA_IMAGES / READ_EXTERNAL_STORAGE: to let users pick handwriting/background images.
